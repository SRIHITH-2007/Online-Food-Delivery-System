# Online Food Delivery System

A DBMS project developed using MySQL for managing customers, restaurants, orders, payments, and delivery services.

## Features
- ER Diagram
- Table Creation with Constraints
- SQL Queries
- Joins and Subqueries
- Views and Triggers
- Stored Procedures

## Technologies Used
- MySQL
- SQL
- DBMS

## Project Modules
- Customer Management
- Restaurant Management
- Order Processing
- Payment Management
- Delivery Partner Management
